  1. Using the functions we wrote during the tutorial, simulate drift over 2000 generations for a population of 100 individuals. How does altering p from 0.3, 0.5 and 0.9 alter the outcome of the simulations? Which is more likely to go to fixation? a custom function for simulating drift across multiple generations
drift_sim <- function(N, p, ngen){
  # initialise p
  p_init <- p
  # sample across all the generations
  pvec <- sapply(1:ngen, function(x){
    pA <- rbinom(1, 2*N, p)
    p <<- pA/(2*N)
  })
  # create a vector of p over time
  p <- c(p_init, pvec)
  # write out
  return(p)
}
# simulate drift for three different p
a <- sapply(c(0.3, 0.5, 0.9), drift_sim, N = 100, ngen = 2000)
